# 🤖 AI Chatbot - Spring Boot + Gemini API

This is a simple AI-powered chatbot built using **Spring Boot** and **Google Gemini API**.  
The application allows users to send messages and receive intelligent responses.

---

## 🚀 Features

- 💬 Chat with AI (Gemini API)
- ⚡ Fast backend using Spring Boot
- 🔗 REST API integration
- 🌐 Easy frontend integration (React / Postman)
- 🔐 Secure API key usage

---

## 🛠️ Tech Stack

- Java
- Spring Boot
- REST APIs
- Gemini API (Google AI)
- Maven
